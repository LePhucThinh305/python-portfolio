# assessment1 – Individual Python Assignment

## Overview
This project consists of four Python programs developed as part of an individual assignment for Introduction to Programming.

The project focuses on writing well-structured functions that perform calculations, validate data, and visualize results using Turtle graphics.

---

## Problem 1 – Household Item Calculator
A function that calculates how many essential items each household will receive during a COVID-19 lockdown based on:

- Number of adults  
- Number of children  
- Number of days in lockdown  
- Allergies  
- Number of adults with medical conditions  

The function:
- Calculates food portions, milk quantities, and toilet paper rolls  
- Rounds all values up  
- Prints a formatted statement  
- Returns all calculated values  

---

## Problem 2 – Bar Chart Visualization
Uses the Turtle module to draw a bar chart displaying:

- Food portions  
- Cow milk  
- Almond milk  
- Treatment milk  
- Toilet paper  

The chart includes labeled axes, quantities, and the household ID.

---

## Problem 3 – Data Validation
Checks for data entry errors including:

- More patients than adults  
- Households with only children  
- Lockdown period less than 14 days  
- Non-integer numeric values  

Returns the household ID and error type while printing descriptive error messages.

---

## Problem 4 – Vaccination Priority Calculator
Determines vaccination priority based on:

- Occupation  
- Age  
- Whether the person lives in a COVID outbreak area  

Returns a dictionary mapping person ID to priority ranking.
